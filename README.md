# NovaFlow × PointClickCare — Compliance Pipeline (interactive demo)

Interactive walkthrough of NovaFlow reading the PDFs buried in PointClickCare's
**Miscellaneous tab**, flagging compliance-sensitive findings against CMS F-Tags,
writing orders back to the chart, and routing tiered alerts (nurse → DON → VP).

Real software in the pipeline: PointClickCare Developer API, Azure Document Intelligence,
medspaCy, PostHog, RAGFlow. Demonstration uses synthetic, de-identified data.

Built by NovaFlow, novaflowapp.com
